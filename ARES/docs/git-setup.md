# Git Setup (Local)

1. Initialize (if not already)
   - `git init`
2. Create a `codex/` branch for your work
   - `git checkout -b codex/initial-setup`
3. Add remote
   - `git remote add origin <your-repo-url>`
4. First commit
   - `git add .`
   - `git commit -m "Initial ARES staging scaffold"`
5. Push
   - `git push -u origin codex/initial-setup`
